CleanOps V7 Device Edition — Real PWA Build

Deployment:
1. Upload this folder to Vercel / Netlify / GitHub Pages.
2. Open the deployed URL in Chrome on the Vivo device.
3. Tap Chrome menu (⋮) → Install app / Add to Home Screen.
4. Launch CleanOps from the home screen.

Included:
- index.html
- manifest.json with display: standalone
- sw.js service worker for offline caching
- app icons
- compact mobile UI for Vivo testing
- staff PIN login, Task/Issue workflow, photo, reports, admin setup, backup/restore
